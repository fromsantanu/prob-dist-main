### ğŸ”¢ **5. Continuous Random Variable**

âœ… **Definition:** Takes **infinite uncountable values** within an interval.

### ğŸ” **6. Probability Distributions for Random Variables**

| Random Variable | Function Type                      |
| --------------- | ---------------------------------- |
| **Discrete**    | Probability Mass Function (PMF)    |
| **Continuous**  | Probability Density Function (PDF) |

---

### ğŸ¯ **7. Example of PMF**

ğŸ‘¤ **Example:** Tossing two coins, define X = number of heads.

| Outcome | X | P(X) |
| ------- | - | ---- |
| HH      | 2 | 1/4  |
| HT      | 1 | 1/4  |
| TH      | 1 | 1/4  |
| TT      | 0 | 1/4  |

âœ… **PMF:**

* P(X=0) = 1/4
* P(X=1) = 2/4 = 1/2
* P(X=2) = 1/4

---

### ğŸ¯ **8. Example of PDF (Conceptual)**

âœ… For continuous variables, **PDF gives density**, not direct probability.

ğŸ”¹ **Example:** Height of students in a class is between 150 cm and 180 cm.

* Probability of exact height = 0
* Probability between 160 cm and 170 cm = Area under PDF curve from 160 to 170
